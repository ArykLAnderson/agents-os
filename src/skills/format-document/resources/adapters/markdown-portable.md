# Markdown Portable Adapter

Use durable local Markdown when the reader needs a repository-readable representation.

## Required Output

- Preserve the source title, section order, decision or recommendation, limitations, and bounded reader action exactly in meaning.
- Keep critical meaning in ordinary headings, paragraphs, and table cells; do not make it available only through a visual, a collapsed structure, or a local tool feature.
- Use stable heading anchors or explicit local anchors for every trace unit that needs a direct locator.
- Keep relative links local and readable. Link to the trace sidecar rather than duplicating provenance in the formatted document.
- When a visual companion exists, link to it and retain the equivalent decision, status, and relationship in adjacent Markdown text.

## Local Evidence

Record that each local link and referenced asset exists, that trace locators resolve, and that a text search finds the critical recommendation, limitation, and reader action.
