# Discovery

Reduce uncertainty in bounded autonomous waves. Frame, not Discovery, decides when knowledge is sufficient.

Use the research shape the current uncertainty needs:

1. **Orient:** supplied material, relevant Cases, maps, repositories, and known sources.
2. **Explore:** broaden enough to expose the landscape and credible alternatives.
3. **Focus:** investigate decision-critical claims and discriminating evidence.
4. **Challenge:** seek contradictions, omitted perspectives, failure modes, and disconfirming evidence.

Apply Frame's clean-context legwork rule to each bounded evidence operation. Run independent gathering in parallel when useful. Centralize synthesis, user interaction, Frame Discovery mutation, deliberation, and Case reconciliation.

Preserve provenance as part of discovery, not as cleanup after synthesis. Every material finding must distinguish direct source support, human authority, and model inference; retain the strongest available source locator plus a pinpoint location or short contextual quotation when available. Record what each source was examined for, preserve material counterevidence and source limitations, and never let a synthesized statement obscure which sources support which part. When reducing a research, prototype, deliberation, review, or modeling artifact to Case entries would lose important argument, method, context, or reproducibility, retain the rich artifact and reference it from the entries instead.

Reuse stable contextual Case knowledge by default. Refresh knowledge that is decision-critical, challenged, volatile, weakly scoped, relitigated, or costly if wrong.

After orientation, after material changes, and before proposing completion, sweep for omitted parties, lifecycle effects, inherited constraints, weak evidence, assumptions, failure modes, and prematurely dismissed alternatives. Persist only material unresolved gaps.

A wave completes only when it:

1. observably reduces or precisely reclassifies the selected uncertainty;
2. updates Frame Discovery;
3. gives every material conclusion an inspectable support chain or marks it unsupported/inferred;
4. preserves important counterevidence and limitations;
5. inventories its material findings, decisions, corrections, and reusable operating lessons;
6. completes Intake, Reconcile, or explicit No Case disposition for each independently reusable unit or cohesive cluster; and
7. preserves detailed evidence outside Cases when condensation would lose argument, method, or provenance.

Only then select the next wave. An active or contested Frame does not defer stable subordinate meaning; classify it provisionally or as contested when that accurately represents its current state.
