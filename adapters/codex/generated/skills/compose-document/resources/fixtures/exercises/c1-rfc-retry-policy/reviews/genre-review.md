# Genre Review

- **Genre:** RFC decision brief
- **Reader:** migration technical approver
- **Outcome:** pass

The decision is first, scope is bounded, the capacity evidence and alternatives are sufficient for the stated action, and the final action distinguishes policy approval from rollout approval. The brief does not imitate a full implementation RFC where the Case has no implementation owner, schedule, or rollout detail.

## Recommendation

Use this shape for the single decision. Add a separate implementation plan only after its owner, schedule, and rollout facts are accepted in the Case.
