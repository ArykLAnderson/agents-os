# Post-Publish Verification

After an authorized write, fetch the destination and inspect the rendered result before declaring publication complete.

- Confirm destination identity, revision, visible current conclusion, limitation, and reader action.
- Confirm block order, child-content preservation, attachment and link resolution, visual alternatives, and narrow-layout readability when the target supports it.
- Capture final page, block, element, and attachment locators in the trace sidecar.
- Record failures as publication blockers. Do not substitute intended locators for fetched locators.
